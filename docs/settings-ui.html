UI & Content Settings

Render forum pages, localize labels, and format values consistently.

These settings shape the public-facing experience of the forum app: layout rendering, translated text, and formatting helpers used in templates and handlers.

Settings

Templates

Use EJS templates with a base layout and optional per-app base overrides. This works well for a forum home page, thread pages, and an admin interface.

templates: {
  enabled: true,
  engine: 'ejs',
  dir: 'templates',
  base: 'base',
  appBases: {
    forums: 'forums/base',
    users: 'users/base',
  },
}

Settings

Internationalization

Forum UIs often need translated menu labels, button text, and validation messages. AegisNode provides locale detection and translation helpers.

i18n: {
  enabled: true,
  defaultLocale: 'en',
  fallbackLocale: 'en',
  supported: ['en', 'fr'],
  translations: {
    en: 'locales/en.json',
    fr: 'locales/fr.json',
  },
}

Settings

Helpers Defaults

Helpers keep date, number, and money formatting consistent throughout the forum UI and admin tools.

helpers: {
  locale: 'en-US',
  money: {
    currency: 'USD',
    currencyDisplay: 'symbol',
  },
}
route.get('/threads/:id', (req, res) => {
  res.render('threads/show', {
    score: req.aegis.helpers.number(15230),
    updatedAgo: req.aegis.helpers.timeElapsed(Date.now() - 3600000),
  });
});
