Overview

AegisNode documentation, organized by feature instead of one long page.

This handbook uses a simple forum app as the example project. Across the pages you will see the same app grow from a fresh scaffold into a modular application with templates, APIs, auth, uploads, mail, and subscribers.

Example Project

The Forum App Used In This Documentation

The docs assume a project named forum. It contains three apps: forums for forum categories, threads for discussions, and users for accounts and moderation helpers. 

export default {
  appName: 'forum',
  apps: [
    { name: 'forums', mount: '/forums' },
    { name: 'threads', mount: '/threads' },
    { name: 'users', mount: '/users' },
  ],
};
You can copy the examples directly and swap forum, forums, or threads for your own app names.
